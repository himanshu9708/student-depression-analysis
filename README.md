# student-depression-analysis
Student depression data analysis with dashboards exploring sleep, study habits, academic pressure, financial stress, and satisfaction levels.

 # Project Objective

The primary objective of this project is to identify patterns, risk factors, and correlations that contribute to student depression.
The analysis focuses on understanding how lifestyle habits (sleep, study), emotional factors (satisfaction), and external pressures (academic load, financial stress) collectively affect student mental health.

- Average Study Hours per Day

## Process
- Collected student data (sleep, study, stress factors).
- Cleaned and organized data for accuracy.
- Analyzed KPIs and patterns.
- Visualized insights using Tableau dashboards.
- Interpreted results to identify depression risk factors.
